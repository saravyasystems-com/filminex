# Roadmap

## Genesis / Sprint 0 — Foundation

- Ratify product charter and vocabulary.
- Approve initial domain boundaries and inheritance rules.
- Choose application stack and repository tooling through an ADR.
- Define tenancy, authentication, authorization, and entitlements.
- Define persistence, asset storage, versioning, and audit events.
- Establish CI, formatting, tests, and local development after stack selection.

## Sprint 1 — Story Workspace

- Workspace and project shell
- Story and read mode
- Episode, scene, shot, and frame hierarchy
- Structured property editor with provenance
- Director notes

## Sprint 2 — Production Libraries

- Character and location libraries
- Reference assets and directional views
- Wardrobe and ornaments
- Camera, lighting, props, expressions, motions, VFX, and music
- Scene assignment and inheritance

## Sprint 3 — Assisted Breakdown

- Script import and structural proposals
- Image-based property proposals
- Review, confidence, correction, and locking
- Provider-independent prompt specification and output

## Later phases

- Collaboration and controlled write access
- AI Studio and animation entitlements
- Direct provider integrations
- Real-time and hybrid production workflows
- Editing and 3D tool integrations
- Billing, operational analytics, and marketplace capabilities

Dates and delivery commitments will be added only after Sprint 0 architecture and staffing decisions.
