# ADR-0020: Governance boundary

- **Status:** Accepted
- **Date:** 2026-07-19

## Context

MTI architecture must remain stable while governance institutions, specifications, and FTRP evolve independently.

## Decision

Future governance includes the MTI Specification, Media Transparency Council, and FTRP. Filminex freezes only their architectural boundaries; it does not freeze council authority, membership, voting, certification, external standardization, or government processes.

Governance documents carry their own maturity labels and never become application architecture merely by appearing in a brainstorm.

## Consequences

- Filminex can implement stable seams without making unsupported institutional claims.
- This ADR remains Accepted until the governance boundary and supporting documents mature sufficiently for a freeze decision.
- Material changes must update or supersede this ADR when frozen later.
